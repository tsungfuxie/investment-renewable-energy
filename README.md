## 產業隱憂與觀察重點

* 供應鏈過於集中於中國市場
  * 分析台灣的產業於全球供應鏈中扮演的腳色
* 太陽能電池製造屬於高汙染產業,回收與處理作業耗費大量成本
  * 尋找有無公司提出改善solution
* 太陽能電池轉換效率技術
  * 跟關鍵材料有關係,目前看來都是透過找到新型材料來提升轉換效率
